
# Strategic Financial Performance Report (2013-2014)

## 📊 Project Overview
This Power BI dashboard provides a comprehensive audit of sales performance, profit margins, and growth trends. 
It is designed to help stakeholders identify top-performing products and regional sales gaps.

## 🚀 Key Features
* **Dynamic KPIs:** Real-time tracking of Total Revenue ($118.7M), Profit Margin (14.23%), and YoY Growth.
* **Top N Analysis:** Interactive filtering to identify the top 5 products by margin.
* **Trend Analysis:** Comparison of current revenue vs. last year (LY) across a fiscal calendar.
* **Performance Status:** An automated logic-based indicator (Excellent/Average/Poor) based on margin thresholds.

## 🛠️ Tools Used
* **Power BI:** Data Visualization & Modeling
* **DAX:** Advanced measures for Time Intelligence and conditional logic
* **Power Query:** Data cleaning and transformation
